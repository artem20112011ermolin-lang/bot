# ⚡ Быстрая настройка (3 шага - 5 минут)

## Шаг 1: Установка зависимостей

```bash
pip install -r requirements.txt
```

Займёт 1-2 минуты.

---

## Шаг 2: Получение токена BotFather

1. Открой **Telegram**
2. Найди бота **@BotFather**
3. Напиши `/newbot`
4. Ответь на вопросы:
   - Имя бота: `Designer Bot` (или любое)
   - Username: `designer_bot_kvod` (должен заканчиваться на `_bot`)
5. **Скопируй полученный токен**

---

## Шаг 3: Получение своего ID

1. Найди бота **@userinfobot**
2. Отправь ему любое сообщение
3. **Скопируй свой ID** (число)

---

## Шаг 4: Заполни .env файл

Открой файл `.env` и вставь:

```env
TELEGRAM_BOT_TOKEN=твой_токен_отсюда
ADMIN_ID=твой_id_отсюда
```

**Пример:**
```env
TELEGRAM_BOT_TOKEN=123456789:ABCDefGhIjKlMnOpQrStUvWxYz
ADMIN_ID=987654321
```

---

## Шаг 5: Запуск! 🚀

```bash
python main.py
```

Если видишь:
```
✅ Менеджер портфолио инициализирован
🤖 Бот запущен!
👤 Admin ID: 987654321
📁 Портфолио: data/portfolio.json
```

**ТЫ УСПЕШНО ЗАПУСТИЛ БОТА!** 🎉

---

## Проверка

1. Открой Telegram
2. Найди своего бота (по username)
3. Отправь `/start`
4. Должны появиться 4 кнопки

---

## Команды для админа

После запуска бота пиши:
- `/add_work` - добавить работу
- `/delete_work` - удалить работу  
- `/list_works` - список работ
- `/admin` - админ-панель

---

## ⛔ Ошибки

**"ModuleNotFoundError"**
```bash
pip install -r requirements.txt
```

**"Invalid token"**
- Скопируй токен заново от @BotFather
- Проверь что нет пробелов

**Бот не отвечает**
- Останови (Ctrl+C) и запусти заново
