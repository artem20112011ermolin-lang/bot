## 🚀 Быстрый старт

```bash

```

### 2. Конфиг .env
```env

ADMIN_ID=твой_id
```

### 3. Запуск

## ❓ FAQ

**Q: Почему без Google Sheets?**  
A: Потому что ты просил удалить - теперь всё проще и локально!

**Q: Где хранятся фото?**  
A: На Telegram серверах. Мы только сохраняем file_id в JSON.

**Q: Потеряются ли работы при перезагрузке?**  
A: На VPS/Hetzner - нет. На Replit бесплатном - возможно (но редко).

**Q: Как сделать нескольких админов?**  
A: Измени в config.py: `ADMIN_ID = [12345, 67890]`

**Q: Что если потеряю file_id?**  
A: Фото остаётся на Telegram, просто добавь работу заново.

**Q: Безопасно ли?**  
A: Да! Все ключи в .env, не отправляются никуда.

**Q: Можно ли добавить БД (типа MongoDB)?**  
A: Да, но пока JSON достаточно. Если надо - скажи.
