## 🏆 РЕКОМЕНДАЦИЯ: Replit (ЛУЧШИЙ ВАРИАНТ)

### Почему Replit?
✅ **Абсолютно бесплатно**  
✅ **Работает 24/7** (уже включено)  
✅ **Ничего не нужно устанавливать** (всё в облаке)  
✅ **5 минут на запуск**  
✅ **Не нужна карта** (условно)  
✅ **Автоматически сохраняет код**

### Пошагово:

#### Шаг 1: Регистрация
1. Перейди на https://replit.com/
2. Нажми **Sign up**
3. Выбери **Sign up with Google** (или любой другой способ)
4. Подтверди почту

#### Шаг 2: Создание проекта
1. Нажми **+ Create** или **New repl**
2. В поле поиска найди **Python**
3. Назови проект (например, "tg_bot_designer")
4. Нажми **Create Repl**

#### Шаг 3: Загрузка кода
1. Нажми на иконку **Upload** (папка со стрелкой) слева
2. **Drag and drop** все файлы из папки `tg_bot` или выбери файлы

Или через Git (если знаешь):
```bash
git clone https://github.com/ТВОЙ_ЮЗЕР/tg_bot.git
cd tg_bot
```

#### Шаг 4: Создание .env файла
1. В левой панели нажми на файл `.env` (или создай новый)
2. Вставь:
   ```env
   TELEGRAM_BOT_TOKEN=123456789:ABCDefGhIjKlMnOpQrStUvWxYz
   ADMIN_ID=987654321
   ```
3. Ctrl+S (сохранить)

#### Шаг 5: Установка зависимостей
1. Открой **Shell** (вкладка внизу) или **Console**
2. Напиши:
   ```bash
   pip install -r requirements.txt
   ```
3. Жди 1-2 минуты

#### Шаг 6: Запуск бота
1. Нажми большую кнопку **Run** (вверху)
2. Должно вывести:
   ```
   ✅ Менеджер портфолио инициализирован
   🤖 Бот запущен!
   ```

**ГОТОВО!** 🎉 Бот работает 24/7 на Replit

### Как проверить?
1. Найди своего бота в Telegram
2. Отправь `/start`
3. Если работает - всё хорошо!

### Продвинутые вещи:
- **Backup кода**: нажми три точки → Export as .zip
- **Поделиться проектом**: нажми Share и отправь ссылку
- **Собственный домен**: нажми Deployments (платно)

## 💎 Альтернатива: PythonAnywhere

### Почему PythonAnywhere?
✅ Бесплатный тариф  
✅ Больше ресурсов чем Replit  
✅ Хороший для Python  
✅ Веб-консоль удобнее

### Как запустить:

1. Перейди на https://www.pythonanywhere.com/
2. **Sign up** (бесплатно)
3. В левом меню нажми **Web app** → **Add a new web app**
4. Выбери **Manual configuration** → **Python 3.10**
5. Загрузи свой код в `/home/твой_юзер/mysite`
6. В консоли:
   ```bash
   cd /home/твой_юзер/mysite
   pip install -r requirements.txt
   python main.py
   ```
7. Готово!

## 🎯 Для опытных: VPS (Hetzner)

### Почему VPS?
✅ Полный контроль  
✅ Дешево ($2-3/мес)  
✅ Работает 24/7  
✅ Один бот на серверечкеhttps://twitter.com/andrewchen/status/1446916180313206790

### Как запустить:

1. **Заказать VPS на Hetzner**
   - Перейди https://hetzner.com/
   - Нажми Cloud → Order
   - Выбери **CPX11** (~2.5€/мес)
   - Выбери OS: **Ubuntu 22.04**
   - Создай сервер

2. **SSH на сервер**
   ```bash
   ssh root@ТВ_IP_АДРЕС
   ```

3. **Установи Python**
   ```bash
   apt update && apt upgrade -y
   apt install python3-pip python3-venv
   ```

4. **Загрузи код**
   ```bash
   git clone https://github.com/твой_юзер/tg_bot.git
   cd tg_bot
   pip install -r requirements.txt
   ```

5. **Создай .env**
   ```bash
   nano .env
   # Вставь переменные
   # Ctrl+O, Enter, Ctrl+X
   ```

6. **Запусти в фоне**
   ```bash
   nohup python3 main.py > bot.log 2>&1 &
   ```

7. **Проверь логи**
   ```bash
   tail -f bot.log
   ```

### Управление:
```bash
# Остановить
pkill -f "python3 main.py"

# Перезагрузить
pkill -f "python3 main.py"
nohup python3 main.py > bot.log 2>&1 &
```

## 📱 Docker (для крутых)

Если знаешь Docker, создай `Dockerfile`:

```dockerfile
FROM python:3.10

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

CMD ["python", "main.py"]
```

Запуск:
```bash
docker build -t tg_bot .
docker run -e TELEGRAM_BOT_TOKEN="токен" -e ADMIN_ID="123" tg_bot
```

## 🆘 Частые проблемы

### Бот не реагирует на команды
- Проверь токен в `.env`
- Убедись, что бот запущен (ищи процесс `python main.py`)
- Перезагрузи бота

### "Port 5000 is already in use"
- Измени порт или убей процесс:
  ```bash
  lsof -i :5000
  kill -9 ПРОЦЕСС_ID
  ```

### Файл portfolio.json потеряется при перезагрузке
- Это нормально на Replit/PythonAnywhere бесплатных тарифах
- Если нужно сохранять, используй базу данных (MongoDB, PostgreSQL)

### Бот работал, а потом остановился
- На бесплатных тарифах иногда спят серверы
- Используй VPS или платные варианты для 100% uptime
